# Deep Research Framework

> Herhangi bir konuda derinlemesine araştırma yaparken kullanılacak sistematik sorgulama çerçevesi.

---

## 1. KONU TANIMLAMA

### 1.1 Problem Tanımı
- [ ] Araştırdığım konu/soru tam olarak nedir?
- [ ] Bu konuyu araştırmamın asıl nedeni ne?
- [ ] Sonunda neyi bilmek/anlamak/çözmek istiyorum?

### 1.2 Kapsam Belirleme
- [ ] **Dahil**: Hangi alt konular dahil?
- [ ] **Hariç**: Neyi bilinçli olarak dışarıda bırakıyorum?
- [ ] **Derinlik**: Yüzeysel mi, teknik detay mı, uzman seviyesi mi?
- [ ] **Zaman dilimi**: Güncel mi, tarihsel süreç mi, gelecek projeksiyonu mu?

### 1.3 Başarı Kriterleri
- [ ] Bu araştırma "başarılı" olduğunda elimde ne olacak?
- [ ] Hangi sorulara cevap bulmuş olacağım?
- [ ] Çıktı formatı ne olmalı? (karar, rapor, uygulama, öğrenme)

---

## 2. ÇOK BOYUTLU SORGULAMA

### 2.1 Temel Sorular (5W2H+)
| Soru | Araştırma Notu |
|------|----------------|
| **NE?** (What) - Tanım, kapsam, bileşenler | |
| **NİÇİN?** (Why) - Sebep, motivasyon, amaç | |
| **NASIL?** (How) - Mekanizma, süreç, yöntem | |
| **KİM?** (Who) - Aktörler, paydaşlar, uzmanlar | |
| **NEREDE?** (Where) - Bağlam, coğrafya, ortam | |
| **NE ZAMAN?** (When) - Zamanlama, tarihçe, dönem | |
| **NE KADAR?** (How much) - Ölçek, maliyet, etki büyüklüğü | |
| **HANGİ KOŞULLARDA?** (Under what conditions) - Bağımlılıklar, ön koşullar | |

### 2.2 Perspektif Değiştirme
Her konuya farklı merceklerden bak:

- [ ] **Tarihsel**: Bu nasıl ortaya çıktı? Evrim süreci ne?
- [ ] **Teknik**: Altta yatan mekanizma/teknoloji ne?
- [ ] **Ekonomik**: Maliyet-fayda, pazar dinamikleri, teşvikler?
- [ ] **Sosyal**: İnsan davranışı, toplumsal etki, kültürel faktörler?
- [ ] **Psikolojik**: Motivasyonlar, önyargılar, karar süreçleri?
- [ ] **Sistemik**: Daha büyük sistemle ilişkisi, geri bildirim döngüleri?
- [ ] **Etik**: Doğru/yanlış, adalet, uzun vadeli sorumluluk?
- [ ] **Pratik**: Gerçek dünyada uygulama, kısıtlar, trade-off'lar?

### 2.3 "Ya ... Olsaydı?" Senaryoları
- [ ] Ya tam tersi doğru olsaydı?
- [ ] Ya bu 10 yıl önce/sonra olsaydı?
- [ ] Ya kaynaklar sınırsız/çok kısıtlı olsaydı?
- [ ] Ya farklı bir kültür/coğrafyada olsaydı?
- [ ] Ya ölçek 10x büyük/küçük olsaydı?

## 3. ELEŞTİREL DÜŞÜNCE TETİKLEYİCİLERİ

### 3.1 Varsayım Sorgulaması
- [ ] Bu konuda **herkesin kabul ettiği** varsayımlar neler?
- [ ] Bu varsayımlardan hangisi **sorgulanabilir**?
- [ ] Bu varsayım **nereden geliyor**? Kanıtı var mı?
- [ ] Varsayım yanlışsa **ne değişir**?

### 3.2 Karşıt Görüş Araştırması
- [ ] Bu konuda **en güçlü karşıt argüman** ne?
- [ ] Akıllı insanlar neden **farklı düşünüyor** olabilir?
- [ ] **Hangi kanıt** beni fikrimden vazgeçirir?
- [ ] Bu görüşün **eleştirmenleri** kim ve ne diyor?

### 3.3 Sınır Durumlar (Edge Cases)
- [ ] Bu **aşırı durumlarda** nasıl çalışır?
- [ ] **İstisnalar** var mı? Neden istisna?
- [ ] **Başarısızlık modları** neler?
- [ ] Sistemin **kırılma noktası** nerede?

### 3.4 İkinci ve Üçüncü Derece Etkiler
- [ ] **Birinci derece**: Doğrudan sonuç ne?
- [ ] **İkinci derece**: O sonucun sonucu ne?
- [ ] **Üçüncü derece**: Beklenmedik yan etkiler?
- [ ] **Uzun vadede** nereye evrilir?

### 3.5 Mantık Hataları Kontrolü
- [ ] **Korelasyon ≠ Nedensellik**: Gerçekten sebep-sonuç mu?
- [ ] **Seçim Yanlılığı**: Sadece başarılı örneklere mi bakıyorum?
- [ ] **Onay Yanlılığı**: Sadece inandığımı destekleyeni mi arıyorum?
- [ ] **Yetki Argümanı**: Uzman dedi diye doğru mu?
- [ ] **Genelleme**: Örneklem temsili mi?

## 4. BİLGİ KALİTESİ DEĞERLENDİRME

### 4.1 Kaynak Güvenilirliği Matrisi

| Kriter | Soru | Değerlendirme |
|--------|------|---------------|
| **Uzmanlık** | Kaynak bu konuda uzman mı? | |
| **Bağımsızlık** | Çıkar çatışması var mı? | |
| **Kanıt** | İddialar kanıtla destekli mi? | |
| **Güncellik** | Bilgi güncel mi? | |
| **Tutarlılık** | Diğer güvenilir kaynaklarla tutarlı mı? | |
| **Şeffaflık** | Metodoloji/kaynak açık mı? | |

### 4.2 Bilgi Türü Ayrımı
- [ ] **Gerçek (Fact)**: Doğrulanabilir, ölçülebilir
- [ ] **Veri**: Ham sayılar, istatistikler
- [ ] **Analiz**: Veriden çıkarılan yorumlar
- [ ] **Görüş (Opinion)**: Kişisel değerlendirme
- [ ] **Spekülasyon**: Tahmin, projeksiyon
- [ ] **Konsensüs**: Uzman topluluğunun genel görüşü

### 4.3 Kaynak Çeşitlendirme
- [ ] **Akademik**: Peer-reviewed makaleler, araştırmalar
- [ ] **Pratisyen**: Saha deneyimi olanlar, uygulayıcılar
- [ ] **Resmi**: Hükümet, kurum raporları
- [ ] **Bağımsız**: Gazeteciler, araştırmacılar
- [ ] **Birincil**: Orijinal veri, doğrudan kaynak
- [ ] **İkincil**: Derleme, analiz, yorum

## 5. SENTEZ VE BAĞLANTI

### 5.1 Disiplinler Arası Bağlantılar
- [ ] Bu konu **hangi disiplinlerle** kesişiyor?
- [ ] Başka alanlardan **hangi kavramlar** uygulanabilir?
- [ ] **Benzer problemler** başka alanlarda nasıl çözülmüş?

### 5.2 Pattern/Örüntü Arama
- [ ] Farklı kaynaklarda **tekrar eden temalar** ne?
- [ ] **Çelişkili bilgiler** var mı? Neden?
- [ ] **Beklenmedik bağlantılar** keşfettim mi?

### 5.3 Analoji ve Metafor
- [ ] Bu konu **neye benziyor**?
- [ ] Hangi analoji en iyi açıklıyor?
- [ ] Analojinin **sınırları** nerede?

### 5.4 Mental Model Eşleştirme
Hangi mental modeller bu konuyu anlamama yardımcı olur?
- [ ] **Pareto (80/20)**: En etkili %20 nedir?
- [ ] **İlk Prensipler**: Temel yapı taşları neler?
- [ ] **Ters Çevirme (Inversion)**: Başarısızlık nasıl olur?
- [ ] **İkinci Derece Düşünme**: Sonra ne olur?
- [ ] **Harita ≠ Arazi**: Model gerçeği ne kadar yansıtıyor?
- [ ] **Fırsat Maliyeti**: Neyi kaçırıyorum?

---

## 6. BOŞLUK ANALİZİ

### 6.1 Bilgi Boşlukları
- [ ] Hala **cevapsız kalan sorular** hangileri?
- [ ] Hangi alanlarda **belirsizlik** var?
- [ ] **Çelişkili bilgilerin** kaynağı ne?

### 6.2 Erişim Kısıtları
- [ ] Hangi kaynaklara **erişemedim**?
- [ ] **Dil/coğrafya** bariyeri var mı?
- [ ] **Paywall/gizlilik** engeli var mı?

### 6.3 Bilmediğimi Bilmediğim Alanlar
- [ ] Bu konuda **neyi bilmediğimi bilmiyorum** olabilir?
- [ ] Hangi **soru sorulmalıydı** ama sormadım?
- [ ] **Kör noktalarım** neler olabilir?

### 6.4 Sonraki Adımlar
- [ ] Araştırmayı **derinleştirmem gereken** alanlar?
- [ ] **Kiminle konuşmalıyım**?
- [ ] **Hangi kaynağı** henüz incelemedim?

## 8. META: ARAŞTIRMA SÜRECİ

### 8.1 Süreç Değerlendirmesi
- [ ] Araştırma **verimli** miydi?
- [ ] **Zaman yönetimi** nasıldı?

- [ ] **Tıkandığım** yerler neresiydi?

### 8.2 Gelecek İçin Notlar
- [ ] Bu tür araştırmalarda **neye dikkat etmeliyim**?

## HIZLI KONTROL LİSTESİ

Araştırmayı bitirmeden önce:

- [ ] Konuyu net tanımladım mı?
- [ ] En az 3 farklı perspektiften baktım mı?
- [ ] Karşıt görüşleri araştırdım mı?
- [ ] Varsayımlarımı sorguladım mı?
- [ ] Kaynakların güvenilirliğini değerlendirdim mi?
- [ ] Bilgi boşluklarını belirledim mi?
- [ ] Pratik çıkarımlar çıkardım mı?
- [ ] Sonraki adımları netleştirdim mi?
